## Key Failure Modes Encountered

### 1. **Shadow-cljs Preload Configuration Issues**
- **Problem**: Preload namespace wouldn't load despite correct syntax
- **Root Cause**: Filename/namespace alignment rules: `dev_preload.cljs` → `evolver.dev-preload`
- **Solution**: Manual REPL loading worked better than fighting preload system
- **Learning**: ClojureScript preloads are fragile; manual REPL approach more reliable

### 2. **Malli Dev Tools ClojureScript Compatibility**
- **Problem**: `malli.dev.cljs/start!` failed with NullPointerException
- **Root Cause**: ClojureScript dev tools have browser/node environment dependencies
- **Solution**: Built custom validation helpers instead of using automatic instrumentation
- **Learning**: Malli's automatic instrumentation works better in Clojure than ClojureScript

### 3. **Console Message Visibility**
- **Problem**: Console logs from ClojureScript REPL not appearing in Chrome DevTools
- **Root Cause**: DevTools message list not refreshing automatically
- **Solution**: Direct JavaScript evaluation showed REPL was connected
- **Learning**: Use `chrome-devtools_evaluate_script` to verify browser state

### 4. **Environment Detection Complexity**
- **Problem**: ClojureScript REPL connection required specific setup order
- **Root Cause**: Browser must be open at localhost:8080 before REPL connection
- **Solution**: Follow MCP-REPL-GUIDE.md connection requirements strictly
- **Learning**: ClojureScript REPL is more environment-sensitive than regular Clojure
